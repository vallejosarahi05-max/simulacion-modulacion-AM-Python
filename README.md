# Simulación de Modulación en Amplitud (AM)

## Descripción
Este proyecto presenta una simulación de un sistema de modulación en amplitud (AM) utilizando Python.  
La simulación permite generar una señal de mensaje, una señal portadora y la señal modulada resultante, además de analizar el efecto del ruido en la señal transmitida mediante diferentes gráficas.

El objetivo es comprender cómo funciona el proceso de modulación en amplitud y cómo las interferencias pueden afectar la calidad de la señal en los sistemas de comunicación.

## Objetivo
Implementar una simulación de modulación en amplitud utilizando Python y las librerías NumPy y Matplotlib para generar y analizar señales en un sistema de comunicación.

## Tecnologías utilizadas
- Python
- NumPy
- Matplotlib

## Funcionamiento del programa
El programa realiza los siguientes procesos:

1. Genera una señal de mensaje de baja frecuencia.
2. Genera una señal portadora de alta frecuencia.
3. Realiza la modulación en amplitud (AM).
4. Agrega ruido a la señal modulada para simular interferencias.
5. Muestra gráficas de las señales generadas para analizar su comportamiento.

## Resultados
El programa genera las siguientes gráficas:

- Señal de mensaje
- Señal portadora
- Señal modulada en amplitud
- Señal modulada con ruido

Estas gráficas permiten observar cómo la señal de mensaje modifica la amplitud de la portadora y cómo el ruido afecta la calidad de la señal transmitida.
